# Experiments

液体GaのS(Q)ショルダー構造再現に関する実験一覧

## 実験一覧

| ID | 実験名 | 説明 | 論文での使用 |
|----|--------|------|-------------|
| 01 | single_component | 単一成分LJモデル | 図3（ショルダー未再現の確認） |
| 02 | bimodal_preliminary | 二成分モデル予備実験 | 未使用（探索範囲の絞り込み用） |
| 03 | bimodal_grid_search | 二成分モデルグリッドサーチ | **図4, 5, 6, 7（主要結果）** |

## 各実験の詳細

### 01_single_component
- 目的: 通常のLJポテンシャルではショルダー構造が再現されないことを確認
- パラメータ: σ = 2.70 Å, ε = 0.430 kcal/mol
- 結果: 対称的なS(Q)ピーク（ショルダーなし）

### 02_bimodal_preliminary
- 目的: 二成分モデルの探索範囲を絞り込むための予備実験
- パラメータ: σ₁₂ = 0.9x, 1.1x、Ga1比率 = 25%, 50%, 75%
- 結果: σ₁₂ > 1.0 でショルダー出現の兆候

### 03_bimodal_grid_search (★論文の主要結果)
- 目的: 最適パラメータの決定
- パラメータ範囲:
  - σ₁₂/σ₁₁: 1.12 〜 1.18（0.01刻み、7段階）
  - Ga1比率: 45% 〜 55%（5段階）
  - 合計: 35通り
- **最適結果**: σ₁₂ = 1.17, Ga1 = 45%, R-factor = 0.058
